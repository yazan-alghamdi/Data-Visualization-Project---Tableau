# Saudi Tourism Analytics Dashboard using Tableau

> An interactive data visualization project that transforms Saudi
> tourism statistics into meaningful dashboards using Tableau to explore
> tourism trends, visitor behavior, regional performance, and spending
> patterns.

------------------------------------------------------------------------

## Overview

This project presents a visual analytics solution for exploring Saudi
tourism statistics (2015--2024). Interactive Tableau dashboards reveal
temporal trends, regional differences, visitor behavior, and tourism
spending to support data-driven decision making.

------------------------------------------------------------------------

## Project Objectives

-   Analyze tourism statistics across Saudi Arabia.
-   Visualize tourism growth over multiple years.
-   Compare domestic and inbound tourism.
-   Explore visitor spending behavior.
-   Identify regional tourism patterns.
-   Build interactive dashboards for decision support.

------------------------------------------------------------------------

## Dataset

The dataset covers Saudi tourism statistics from **2015--2024** and
includes:

-   Number of Tourists
-   Tourist Spending
-   Average Stay Duration
-   Average Spending per Trip
-   Average Spending per Night
-   Province
-   Tourism Type
-   Year

------------------------------------------------------------------------

## Dashboard Features

-   Tourism Trend Line Chart
-   Stacked Bar Chart
-   Geographic Map
-   Spending Heatmap
-   Spending Distribution Box Plot
-   Spending vs Stay Scatter Plot

## Key Insights

-   Makkah records the highest tourism activity.
-   Riyadh and Eastern Province are major tourism hubs.
-   Tourism declined during 2020--2021 before recovering strongly.
-   Longer visitor stays generally lead to higher spending.
-   Domestic tourism dominates earlier years while inbound tourism
    continues to grow.
